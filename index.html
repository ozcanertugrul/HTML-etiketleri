Liste Etiketleri

1. Sıralı Liste (<ol>):

HTML öğrenirken takip edilecek adımlar:

  1. HTML etiketlerini öğrenin.
  2. Sayfa yapısını oluşturun.
  3. Stil ve tasarım ekleyin (CSS).
  4. Dinamik içerik ekleyin (JavaScript).

2. Tanım Listesi (<dl>, <dt>, <dd>):

HTML terimleri ve açıklamaları:

HTML
HyperText Markup Language'in kısaltmasıdır ve web sayfalarını oluşturmak için kullanılır.
CSS
Web sayfalarına stil eklemek için kullanılan bir tasarım dilidir.
JavaScript
Web sayfalarını dinamik hale getiren bir programlama dilidir.

Bağlantı Etiketleri

1. Harici Kaynak Bağlantısı (<link>):

Aşağıda sayfamıza bir CSS dosyasını bağlayan `` etiketi örneği bulunmaktadır:

                <link rel="stylesheet" href="styles.css">

Bu sayede harici bir CSS dosyasından stiller ekleyebilirsiniz.

2. İçerik Bağlantısı (<a>):

Aşağıdaki bağlantıları kullanarak diğer kaynaklara veya sayfa içindeki bir bölüme gidebilirsiniz:

  • Google'ı ziyaret edin
  • Hakkında bölümüne git
  • Bize e-posta gönder
